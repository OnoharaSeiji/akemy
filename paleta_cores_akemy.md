# Paleta funcional AKEMY

Objetivo: melhorar legibilidade, organização e retenção sem depender de decoração excessiva.

## Função de cada cor
- Azul profundo `#243B6B`: títulos, botões principais, conceitos
- Verde `#2F6F5E`: exemplos, aplicação, progresso
- Vermelho terroso `#A84B4B`: alerta, erro comum, atenção
- Bege claro `#E9D8A6`: dicas, fórmulas, caixas de apoio
- Texto principal `#1F2937`
- Fundo principal `#F7F6F2`
- Cards `#FFFFFF`

## Regras de uso
- usar poucas cores por tela
- manter a mesma cor para a mesma função
- evitar blocos longos com fundo muito escuro
- priorizar contraste alto
- não depender só de cor para significado

## O que melhora
- leitura longa
- escaneabilidade
- hierarquia visual
- redução de fadiga
- consistência entre home e apostilas
